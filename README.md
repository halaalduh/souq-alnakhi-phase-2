# Souq Al-Nakhil 🌴

Souq Al-Nakhil is a web-based agricultural marketplace that connects local Saudi farmers with consumers. It provides a centralized platform for browsing, searching, and managing date products with a focus on transparency and accessibility.

## Overview

The platform addresses the lack of digital solutions for small-scale farmers and the difficulty consumers face in finding reliable local products.

## Features

- User registration, login, and profile management  
- Farm profile creation and product management (add, edit, delete)  
- Product browsing and search  
- Filtering by region and date type  
- Viewing product and farm details  
- Admin verification and trusted farmer badge  

## Architecture

The system follows a client–server architecture:
- Client: Web interface  
- Server: Business logic and processing  
- Database: Stores users, farms, and products   

## Scope

The system is a web application and does not include:
- Online payment  
- Delivery services  
- Mobile application
